# GetOutputNum（MetaContext类）

## 产品支持情况

- Atlas A3 训练系列产品/Atlas A3 推理系列产品：支持
- Atlas A2 推理系列产品/Atlas A2 训练系列产品：支持

## 函数功能

获取Flowfunc的输出个数。该函数供[Init（MetaFlowFunc类）](init-metaflowfunc-class.md)调用。

## 函数原型

```cpp
size_t GetOutputNum() const
```

## 参数说明

无

## 返回值

返回Flowfunc的输出个数。

## 异常处理

无。

## 约束说明

无。
