# 学习向导

## 读者对象

本文档用于指导开发者如何使用DataFlow接口进行异步流水应用的开发，或者对原应用进行异步流水的改造，实现应用的大吞吐和高并发。通过本文档您可以达成以下目标：

- 了解DataFlow的基本概念和使用场景。
- 了解使用DataFlow接口进行应用开发的基本流程和实现方法。
- 能够基于本文档中的样例，扩展进行其他应用的开发。

熟悉CANN软件基本架构以及特性、具备Python语言程序开发能力，对机器学习、深度学习有一定了解的人员，可以更好地理解本文档。
