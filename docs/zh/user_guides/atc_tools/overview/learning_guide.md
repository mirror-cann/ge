# 学习向导

本文档用于指导开发者如何使用ATC（Ascend Tensor Compiler，简称ATC）工具进行模型转换，得到适配AI处理器的离线模型。通过本文档您可以达成以下目标：

- 了解不同框架原始网络模型转成AI处理器离线模型的方法。
- 能够基于本文档中的参数，转成满足不同定制要求的离线模型。

熟悉Linux基本命令，对机器学习、深度学习有一定了解的人员，可以更好地理解本文档。
