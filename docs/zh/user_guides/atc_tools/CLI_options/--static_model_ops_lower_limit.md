# --static\_model\_ops\_lower\_limit

## 产品支持情况

全量芯片支持

## 功能说明

**动态shape拆分中，设置静态子图**最小算子数量阈值。

## 关联参数

无。

## 参数取值

**参数值：**
\[-1, INT64\_MAX\]范围内的整数，取其他值报错。

- 取值\>=0：
  - 算子数<门限，静态子图不单独拆图，合并到动态图走动态执行。
  - 算子数\>=门限，保持原有流程不变（静态子图仍旧会拆图）。

- 取值=-1：表示所有算子走动态图执行。

例如，假设静态子图中算子数为4，通过该参数配置的门限为10，则静态子图不会单独拆图，走动态图执行。

**参数默认值**：4

## 推荐配置及收益

无。

## 示例

```bash
atc --static_model_ops_lower_limit=4 --model=model.onnx ...
```

## 依赖约束

无。
