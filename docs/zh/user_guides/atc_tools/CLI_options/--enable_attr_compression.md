# --enable\_attr\_compression

## 产品支持情况

全量芯片支持

## 功能说明

是否对图中的属性进行压缩。

使用ATC工具生成离线模型时，会自动对图中的属性进行压缩，以减小模型体积；但在部分场景下，压缩后的属性解压过程耗时较高，可能影响推理性能；为此，增加该参数，用于控制是否在生成离线模型时对图属性进行压缩。

## 参数取值

- false：生成离线模型时，不压缩图中的属性。
- true：（默认值）生成离线模型时，自动压缩图中的属性。

## 推荐配置及收益

无。

## 示例

```bash
atc --enable_attr_compression=true ...
```
