# --oo\_dead\_code\_elimination

## 产品支持情况

全量芯片支持

## 功能说明

**调试功能扩展参数，暂不支持应用于生产环境，后续版本会作为正式功能更新发布。**

是否开启死边消除优化。

死边消除优化：在Switch节点中，若其第一个输入（pred输入）为常量节点（const），则根据该常量的值消除冗余分支。若常量为true，则移除false分支；若常量为false，则移除true分支。

## 关联参数

无。

## 参数取值

- true：（默认值）开启死边消除优化。
- false：关闭死边消除优化。

## 推荐配置及收益

无。

## 示例

```bash
atc --oo_dead_code_elimination=true ...
```

## 使用约束

无。
