# --model

## 产品支持情况

全量芯片支持

## 功能说明

原始网络模型文件路径与文件名。

## 关联参数

当原始模型为Caffe框架时，需要和[--weight](--weight.md)参数配合使用。

## 参数取值

**参数值：**
模型文件路径与文件名。

**参数值格式：**
路径和文件名：支持大小写字母（a-z，A-Z）、数字（0-9）、下划线（\_）、短横线（-）、句点（.）、中文汉字。

## 推荐配置及收益

无。

## 示例

```bash
atc --model=$HOME/module/resnet50_tensorflow.pb ...
```

## 依赖约束

无。
