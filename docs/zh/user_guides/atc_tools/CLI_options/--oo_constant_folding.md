# --oo\_constant\_folding

## 产品支持情况

全量芯片支持

## 功能说明

**调试功能扩展参数，暂不支持应用于生产环境，后续版本会作为正式功能更新发布。**

是否开启常量折叠优化。

常量折叠是将计算图中可以预先确定输出值的节点替换成常量，并对计算图进行一些结构简化的操作。

## 关联参数

无。

## 参数取值

**参数值：**

- true：（默认值）开启常量折叠优化。
- false：关闭常量折叠优化。

## 推荐配置及收益

无。

## 示例

```bash
atc --oo_constant_folding=true ...
```

## 使用约束

无
