## 功能说明<a name="section187304428437"></a>

根据模型描述信息获取模型的输入tensor的维度范围。

## 函数原型<a name="section18474205015436"></a>

```
aclError aclmdlGetInputDimsRange(const aclmdlDesc *modelDesc, size_t index, aclmdlIODimsRange *dimsRange)
```
