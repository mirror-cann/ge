## 功能说明<a name="section44282627"></a>

## 函数原型<a name="section1831611148519"></a>

```
std::unique_ptr<Tensor> CreateBoolTensor(const std::vector<bool> &value, const std::vector<int64_t> &dims, Format format = FORMAT_ND)
```
