## 头文件/库文件<a name="section710017525388"></a>

-   头文件：\#include <graph/graph.h\>
-   库文件：libgraph.so

## 功能说明<a name="section145571943181415"></a>

将序列化后的Graph保存到内存中。

## 函数原型<a name="section1416444114149"></a>

```
graphStatus SaveToMem(GraphBuffer &graph_buffer) const
```
