## 功能说明<a name="section259105813316"></a>

模型执行阶段，如果涉及动态更新变量的场景，可以调用本接口从模型文件获取其描述信息。

## 函数原型<a name="section2067518173415"></a>

```
aclError aclmdlBundleQueryInfoFromFile(const char* fileName, aclmdlBundleQueryInfo *queryInfo)
```
