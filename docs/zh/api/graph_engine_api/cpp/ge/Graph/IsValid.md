# IsValid

## 产品支持情况

全量芯片支持。

## 头文件/库文件

- 头文件：\#include <graph/graph.h\>
- 库文件：libgraph.so

## 功能说明

判断Graph对象是否有效，即是否可被运行。

## 函数原型

```c++
bool IsValid() const
```

## 参数说明

无

## 返回值说明

| 参数名 | 类型 | 说明 |
| --- | --- | --- |
| - | bool | - True，构建Graph对象有效，可被执行。<br>  - False，构建Graph对象无效，不可被执行。 |

## 约束说明

无
