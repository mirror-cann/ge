# AddNodeByOp

## 产品支持情况

全量芯片支持。

## 头文件/库文件

- 头文件：\#include <graph/graph.h\>
- 库文件：libgraph.so

## 功能说明

新增一个节点到图上。

## 函数原型

```c++
GNode AddNodeByOp(const Operator &op)
```

## 参数说明

| 参数名 | 输入/输出 | 说明 |
| --- | --- | --- |
| op | 输入 | 新增节点的原型op对象。 |

## 返回值说明

| 参数名 | 类型 | 说明 |
| --- | --- | --- |
| - | GNode | 返回新增的节点。 |

## 约束说明

此时算子未连接边，该node未生效，需要显式补充连边关系。
