# GetAllSubgraphs

## 产品支持情况

全量芯片支持。

## 头文件/库文件

- 头文件：\#include <graph/graph.h\>
- 库文件：libgraph.so

## 功能说明

获取所有的子图。

## 函数原型

```c++
std::vector<GraphPtr> GetAllSubgraphs() const
```

## 参数说明

无

## 返回值说明

| 参数名 | 类型 | 说明 |
| --- | --- | --- |
| - | std::vector\<GraphPtr\> | 获取的所有子图。 |

## 约束说明

无
