# SetPassName

## 产品支持情况

全量芯片支持。

## 头文件/库文件

- 头文件：\#include <register/register\_custom\_pass.h\>
- 库文件：libregister.so

## 功能说明

设置自定义Pass的名称。

## 函数原型

```c++
void SetPassName(const AscendString &pass_name)
```

## 参数说明

| 参数名 | 输入/输出 | 说明 |
| --- | --- | --- |
| pass_name | 输入 | 要设置的Pass名称。 |

## 返回值说明

无

## 约束说明

无
