# IMPLEMT\_VERIFIER

## 产品支持情况

全量芯片支持。

## 头文件

\#include <graph/operator\_reg.h\>

## 功能说明

封装算子的Verify函数。

## 函数原型

```c++
IMPLEMT_VERIFIER(op_name, func_name)
```

## 参数说明

| 参数名 | 输入/输出 | 描述 |
| --- | --- | --- |
| op_name | 输入 | 算子类型。 |
| func_name | 输入 | Verify函数名，用户自定义。 |

## 返回值说明

无

## 约束说明

无
