# GetSize

## 产品支持情况

全量芯片支持。

## 头文件/库文件

- 头文件：\#include <graph/graph\_buffer.h\>
- 库文件：libgraph\_base.so

## 功能说明

获取序列化后的Graph的内存的长度。

## 函数原型

```c++
std::size_t GetSize() const
```

## 参数说明

无

## 返回值说明

| 参数名 | 类型 | 说明 |
| --- | --- | --- |
| - | std::size_t | 序列化后的Graph的内存的长度。 |

## 约束说明

无
