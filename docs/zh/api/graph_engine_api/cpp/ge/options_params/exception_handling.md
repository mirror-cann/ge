# 异常补救

## event\_sync\_timeout

图执行时，event同步等待超时时间，超过配置时间时报同步失败。单位：ms

默认值-1，表示无等待时间，出现同步失败不报错。

**配置示例：**

```c++
{"event_sync_timeout", "-1"};
```

**必选/可选**：可选

**生效级别**：全局/session
