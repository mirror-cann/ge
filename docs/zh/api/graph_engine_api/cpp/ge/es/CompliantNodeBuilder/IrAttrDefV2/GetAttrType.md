# GetAttrType

## 产品支持情况

全量芯片支持。

## 头文件/库文件

- 头文件：\#include <ge/compliant\_node\_builder.h\>
- 库文件：libeager\_style\_graph\_builder\_base.so、libeager\_style\_graph\_builder\_base\_static.a

## 功能说明

获取属性类型。

## 函数原型

```c++
IrAttrType GetAttrType() const
```

## 参数说明

无

## 返回值说明

| 参数名 | 类型 | 说明 |
| --- | --- | --- |
| - | IrAttrType | 返回通过[AttrType](AttrType.md)设置的属性类型枚举值。 |

## 约束说明

无
